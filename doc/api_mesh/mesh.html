CGL.Mesh

new Mesh(cgl: Context, geometry: Geometry, glPrimitive: Number?)

webgl renderable 3d object


Example:
const cgl=op.patch.cgl
const mesh=new CGL.Mesh(cgl, geometry);

function render()
{
  mesh.render(cgl.getShader());	
}
