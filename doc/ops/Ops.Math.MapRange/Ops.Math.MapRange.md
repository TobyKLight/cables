# MapRange

*Ops.Math.MapRange*  

Maps a value from one range into another.

## Input

### Value

*Type: Value*  
The value you want to map

### Old Min

*Type: Value*  
The minimum of the old range

### Old Max

*Type: Value*  
The maximum of the old range

### New Min

*Type: Value*  
The minimum of the new range

### New Max

*Type: Value*  
The maximum of the new range

## Output

### Result

*Type: Value*  
Result of the mapping
