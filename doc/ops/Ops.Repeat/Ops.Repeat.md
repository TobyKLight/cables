# Repeat

*Ops.Repeat*

Triggers all ops underneath `num` times

## Input

### Exe

*Type: Function*

Triggers repeated execution

### Num

*Type: Value*

Number of times childs should be executed

## Output

### Trigger

*Type: Function*

Triggers execution of childs
