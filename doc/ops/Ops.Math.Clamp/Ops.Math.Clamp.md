# Clamp

*Ops.Math.Clamp*  

Makes sure a value is within range, cuts off the rest.

## Input

### Value

*Type: Value*  
The number you want to make sure is in range [`Minimum`, `Maximum`]

### Minimum

*Type: Value*  
The minimum the `Result` will be

### Maximum

*Type: Value*  
The maximum the `Result` will be

## Output

### Result

*Type: Value*  
The clamped value of `Value`
