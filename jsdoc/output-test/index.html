PatchConfig

patch-config

PatchConfig

Type: Object

Parameters
prefixAssetPath (String = '') path to assets
glCanvasId (String = 'glcanvas') dom element id of canvas element
onError (Function = null) called when an error occurs
onFinishedLoading (Function = null) called when patch finished loading all assets
onFirstFrameRendered (Function = null) called when patch rendered it's first frame
glCanvasResizeToWindow (Boolean = false) resize canvas automatically to window size
doRequestAnimation (Boolean = true) do requestAnimationFrame set to false if you want to trigger exec() from outside (only do if you know what you are doing)
clearCanvasColor (Boolean = true) clear canvas in transparent color every frame
clearCanvasDepth (Boolean = true) clear depth every frame
silent (Boolean = false)
fpsLimit (Number = 0) 0 for maximum possible frames per second
glslPrecision (String = 'mediump') default precision for glsl shader
