# Blur

*Ops.Gl.TextureEffects.Blur*

![blur](img/blur.jpg)


Blur the current state of an image compose

## Input

### Render

*Type: Function*

Trigger the rendering of the effect

### Amount

*Type: Value*

Amount of blur (0 - no blur)

## Output

### trigger

*Type: Function*

Trigger the next image effect
