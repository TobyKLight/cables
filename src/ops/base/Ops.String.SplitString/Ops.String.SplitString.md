# SplitString

*Ops.String.SplitString.js*  

Splits a string like `"111,112,113,114"` into single values `"111"`, `"112"`, `"113"`, `"114"`.  

## Input 

### Input String [String]

The string you want to split

### Separator [String]

The separator char, e.g. `,`	

## Output

### Part 1-8

The single parts of the string
