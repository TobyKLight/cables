# ScaleXYZ

*Ops.Gl.Matrix.ScaleXYZ*

Enlarges or shrinks objects by a scale factor for every axis (non-uniform scaling)


## Input

### Render

*Type: Function*

Triggers the scaling

### X

*Type: Value*

Scale factor X

### Y

*Type: Value*

Scale factor Y

### Z

*Type: Value*

Scale factor Z

## Output

### Trigger

*Type: Function*

Triggers the next operator
