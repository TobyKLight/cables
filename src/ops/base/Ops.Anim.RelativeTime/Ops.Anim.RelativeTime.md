# RelativeTime

*Ops.Anim.RelativeTime*

Returns the time the patch is running in seconds, e.g. `22.567` = 22 seconds and 567 milliseconds.

## Input

### Execute

*Type: Function*
Triggers the *RelativeTime*-op

## Output

### Result

*Type: Function*  
The time the patch is running (in seconds), e.g. `22.567`
