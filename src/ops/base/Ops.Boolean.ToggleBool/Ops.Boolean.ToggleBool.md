# Ops.ToggleBool

Outputs a bool value which changes state every time `Trigger` is executed, this will result in an output like `true` -> `false` -> `true` -> `false` â€¦  

## Input

### Trigger

*Type: Function*  
When triggered the output value will be toggled from `true` to `false` and the other way around.

## Output

### Result

*Type: Value [true, false]*
