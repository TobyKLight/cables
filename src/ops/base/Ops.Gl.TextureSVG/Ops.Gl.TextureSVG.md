# TextureSVG

*Ops.Gl.TextureSVG*

Loads a SVG images and convert them to a texture.

## Input

### file

*Type: Value*

SVG File URL

### Texture width

*Type: Value*

Width of the texture

### Texture height

*Type: Value*

Height of the texture


## Output

### Texture

*Type: Object*

Resulting texture
