# ColorPalettes

*Ops.Color.ColorPalettes*

a collection of nice color palettes of 5 colors each.

## Input

### Index [Value]
The color pallet you want to get (between 0 and 999)

## Output

### Color Array [Array]

An array with 15 values (5 colors with red, green and blue values).

0: Color 1 Red
1: Color 1 Green
2: Color 1 Blue

3: Color 2 Red
4: Color 2 Green
5: Color 2 Blue

6: Color 3 Red
7: Color 3 Green
8: Color 3 Blue

9: Color 4 Red
10: Color 4 Green
11: Color 4 Blue

12: Color 5 Red
13: Color 5 Green
14: Color 5 Blue

You can use the *ArrayGetValue*-op to access the individual colors
