# Concat

*Ops.String.concat*  

Joins two strings together, e.g. `"foo"` + `"bar"` = `"foobar"`

## Input

### String1

*Type: Value [String]*  
Beginning of the resulting string

### String2

*Type: Value [String]*  
End of the resulting string

## Output

### Result

*Type: Value*  
`String1` + `String2`
